# kali-linux-home-lab
Beginner cybersecurity lab using Kali Linux

I taught myself how to set up a personal cybersecurity lab using Kali Linux on a virtual machine called UTM. I researched and practiced basic networking and system commands.

The tools that I used:
- Kali Linux
- Nmap
- Linux Terminal

Lab Activity 1: Basic Network Scan

The Command: nmap localhost
- I learned that this command scans your own computer to check for open ports and running services.

- The result of this cammand was all 1000 common ports are closed and no active services were running.
  
What I Learned: 
How to set up a virtual machine,
basic cybersecurity tools,
and the basics of network scanning.


Lab Activity 2: Network Information

Command Used: ifconfig

- I learned that this command displays network interface details like IP address, MAC address, and packet activity.
- The result of this command was found network interface , viewed IP, and system network details.
  
What I Learned:
How to find system network information and understanding how devices connect to a network.

Lab Activity 3: Service Version Detection

Command: nmap -sV localhost

- I learned how to run a more detailed scan to try to detect services and versions running on ports.
- The result of this command was there was no service found and everything was closed.

What I Learned:
How to gather more detailed information from scans and how attackers might look for services.

Lab Activity 4: System Shutdown

Command: sudo shutdown now

- I learned how to safely shut down the system using admin privileges.
- I learned a sudo command and the importance of properly shutting down a system.

The Overall Skills I Gained:
- Set up a virtual cybersecurity lab
- Used Linux terminal commands
- Performed basic network scanning and analysis
- Practiced commands in a controlled virtual lab environment
- Practiced safe system management
